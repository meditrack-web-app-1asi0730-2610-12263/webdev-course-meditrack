# Checklist final

Antes de terminar tu página, revisa lo siguiente:

- [ ] El archivo principal se llama `index.html`.
- [ ] Las etiquetas HTML están correctamente abiertas y cerradas.
- [ ] El archivo `styles.css` está conectado con `link`.
- [ ] Las imágenes tienen una ruta o URL válida.
- [ ] Los textos se leen con claridad.
- [ ] La página se puede abrir en el navegador.
