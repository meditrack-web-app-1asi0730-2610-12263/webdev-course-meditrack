# Guion de video 3: Elementos HTML comunes

## Objetivo
Enseñar listas, enlaces e imágenes.

## Estructura sugerida

1. Explicar que una página puede tener diferentes tipos de contenido.
2. Crear una lista con `ul` y `li`.
3. Crear un enlace con `a`.
4. Insertar una imagen con `img` y explicar el uso de `alt`.
5. Probar el ejemplo terminado.
6. Cierre: ahora la página tiene más contenido y será momento de darle estilo.

## Frase de práctica
Agrega tres intereses personales y un enlace a una página educativa.
