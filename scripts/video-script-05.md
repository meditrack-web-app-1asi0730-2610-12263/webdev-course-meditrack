# Guion de video 5: Página web personal

## Objetivo
Construir una página de perfil personal combinando HTML y CSS.

## Estructura sugerida

1. Presentar el resultado final esperado.
2. Crear una tarjeta principal con `main`.
3. Agregar título, descripción, intereses y enlace.
4. Aplicar estilos para ordenar el contenido.
5. Revisar el resultado en el navegador.
6. Cierre: el estudiante ya creó su primera página completa.

## Frase de práctica
Personaliza el texto, los intereses y el enlace final.
