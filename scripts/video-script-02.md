# Guion de video 2: Primera estructura HTML

## Objetivo
Mostrar las partes principales de un documento HTML.

## Estructura sugerida

1. Recordar que HTML organiza el contenido.
2. Explicar `DOCTYPE`, `html`, `head` y `body` sin usar términos complejos.
3. Agregar un `h1` para el título principal.
4. Agregar un `p` para una descripción breve.
5. Ejecutar el resultado en un editor online.
6. Cierre: HTML permite agregar más tipos de contenido.

## Frase de práctica
Escribe un título y un párrafo sobre un tema que te guste.
