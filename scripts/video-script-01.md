# Guion de video 1: ¿Qué es el desarrollo web?

## Objetivo
Explicar de forma sencilla qué es una página web y para qué sirven HTML y CSS.

## Estructura sugerida

1. Saludo breve.
2. Explicación: una página web es un documento que el navegador puede mostrar.
3. Comparación simple: HTML es el contenido y CSS es la apariencia.
4. Mostrar el archivo `starter-files/lesson-01/index.html`.
5. Pedir al estudiante que cambie el texto del título y del párrafo.
6. Cierre: en la siguiente lección se revisará la estructura HTML.

## Frase de práctica
Pausa el video y cambia el título por uno creado por ti.
