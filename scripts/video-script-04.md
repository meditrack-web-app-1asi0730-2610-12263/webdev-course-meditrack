# Guion de video 4: Introducción a CSS

## Objetivo
Aplicar estilos básicos a una página HTML.

## Estructura sugerida

1. Explicar que CSS cambia la apariencia del contenido.
2. Conectar `styles.css` con la etiqueta `link`.
3. Cambiar el color de fondo del `body`.
4. Crear una tarjeta usando la clase `.card`.
5. Agregar borde, espaciado y alineación.
6. Cierre: CSS ayuda a que la página sea más clara y agradable.

## Frase de práctica
Cambia el color del título y agrega un borde a la tarjeta.
