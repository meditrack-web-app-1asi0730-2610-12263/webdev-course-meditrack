# Fundamentos de Desarrollo Web

## Resumen del curso

Este curso introduce a estudiantes de secundaria a los fundamentos del desarrollo web mediante HTML y CSS. Durante aproximadamente una hora, los estudiantes aprenderán qué es una página web, cómo se estructura con HTML, cómo se agregan elementos comunes y cómo se mejora visualmente con CSS. Al final, crearán una página de perfil personal sencilla.

El curso está diseñado para principiantes absolutos, por lo que se evita el uso de términos complejos y se utilizan ejemplos visuales, pausas para práctica y actividades guiadas.

- **Duración total:** ~60 minutos
- **Público objetivo:** estudiantes de 12 a 17 años
- **Experiencia previa requerida:** ninguna
- **Herramientas necesarias:** navegador web y editor online gratuito
- **Repositorio:** https://github.com/meditrack-web-app-1asi0730-2610-12263/webdev-course-meditrack

## Herramientas utilizadas

- CodePen: para prácticas rápidas de HTML y CSS.
- JSFiddle: para probar ejemplos pequeños.
- Replit: para el proyecto final.
- YouTube: para alojar las lecciones como videos no listados.

## Secuencia de lecciones

### Lección 1: ¿Qué es el desarrollo web? (8 minutos)

- **Descripción:** Se explica qué es un sitio web y cómo HTML y CSS trabajan juntos.
- **Video:** [Ver lección 1](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_1)
- **Consejo clave:** HTML organiza el contenido y CSS define el estilo visual.
- **Práctica:** [Abrir práctica en CodePen](https://codepen.io/pen/)
- **Archivo inicial:** `starter-files/lesson-01/index.html`

### Lección 2: Primera estructura HTML (10 minutos)

- **Descripción:** Se crea una primera página con `html`, `head`, `body`, `h1` y `p`.
- **Video:** [Ver lección 2](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_2)
- **Consejo clave:** Todo documento HTML debe tener una estructura clara.
- **Práctica:** [Abrir práctica en CodePen](https://codepen.io/pen/)
- **Archivo inicial:** `starter-files/lesson-02/index.html`

### Lección 3: Elementos HTML comunes (10 minutos)

- **Descripción:** Se agregan listas, enlaces e imágenes para enriquecer la página.
- **Video:** [Ver lección 3](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_3)
- **Consejo clave:** Las etiquetas permiten representar diferentes tipos de contenido.
- **Práctica:** [Abrir práctica en JSFiddle](https://jsfiddle.net/)
- **Archivo inicial:** `starter-files/lesson-03/index.html`

### Lección 4: Introducción a CSS (12 minutos)

- **Descripción:** Se aplican colores, fuentes, bordes y centrado de contenido.
- **Video:** [Ver lección 4](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_4)
- **Consejo clave:** Los selectores CSS indican qué elementos recibirán estilo.
- **Práctica:** [Abrir práctica en CodePen](https://codepen.io/pen/)
- **Archivo inicial:** `starter-files/lesson-04/index.html`

### Lección 5: Creación de una página web personal (15 minutos)

- **Descripción:** Se combinan HTML y CSS para crear una página de perfil.
- **Video:** [Ver lección 5](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_5)
- **Consejo clave:** Un proyecto pequeño ayuda a practicar estructura, contenido y estilo.
- **Proyecto final:** [Abrir proyecto en Replit](https://replit.com/)
- **Archivo inicial:** `starter-files/lesson-05/index.html`

### Lección 6: Errores comunes y próximos pasos (5 minutos)

- **Descripción:** Se revisan errores frecuentes y recomendaciones para seguir aprendiendo.
- **Video:** [Ver lección 6](https://www.youtube.com/watch?v=REEMPLAZAR_LECCION_6)
- **Consejo clave:** Revisar etiquetas, rutas de archivos y nombres ayuda a resolver errores.
- **Checklist:** `starter-files/lesson-06/checklist.md`

## Recomendaciones para los videos

- Usar una resolución legible para que el código se vea claramente.
- Explicar con lenguaje sencillo y ejemplos cotidianos.
- Hacer pausas cortas para que el estudiante practique.
- Publicar los videos como no listados en YouTube.
- Colocar en la descripción de cada video el enlace al editor online correspondiente.

## Cierre

Al finalizar el curso, el estudiante habrá creado una página web básica de perfil personal y comprenderá la diferencia entre HTML y CSS.
